이 문서는 바둑 프로그램 개요, 인터페이스 및 기능, 소스 파일을 설명한 기술 문서이다.

1. 프로그램 개요
두 명의 사용자가 오목을 두는 프로그램

2. 프로그램 인터페이스(기능)
2.1. 바둑판
바둑판은 점과 선으로 이루어진 바둑판과 바둑판의 좌표, 커서, 착수된 바둑알로 이루어진다.
바둑판의 크기는 레귤러(15×15), 점보(19×19) 두 가지가 있다.

2.1.1. 선으로 이루어진 바둑판과 바둑판의 좌표
선으로 이루어진 바둑판과 바둑판의 좌표의 예시다.
바둑판의 선을 이루는 문자(┌ ┬ ┐ └ ┴ ┘ ├ ─ ┤)는 'ㅂ'을 누르고 한자 키를 누르면 입력할 수 있고,
점 문자(·)는 'ㄱ' 을 누르고 한자키를 누르면 입력할 수 있다.

바둑판의 좌표는 알파벳과 숫자로 이루어져 있다.
알파벳은 행의 위치를 나타내고, 숫자는 열의 위치를 나타낸다.
행의 위치는 A부터 시작하고, 열의 위치는 1부터 시작한다.

2.1.2. 커서
바둑알을 착수할 위치는 키보드(WASD 또는 방향키)를 통해 커서를 옮겨 선택할 수 있다.
커서를 표현하는 문자는 흑의 차례는 ◐, 백의 차례는 ◑으로 한다.
이 문자는 'ㅁ'을 누르고 한자키를 눌러 입력할 수 있다.

3. 소스 파일 설명
3.1. omokobj.c / omokobj.h
오목 프로그램에서 사용되는 모든 객체(구조체)를 정의
[객체 종류]
-좌표정보 객체(행과 열 정보)
-바둑알 정보 객체(바둑알 종류)
-커서 객체(바둑알 정보, 좌표정보)
-바둑알 객체(바둑알 정보, 좌표정보)
-바둑알 저장 자료구조 객체(연결 리스트, 배열 등)
-바둑판 객체(바둑판 크기, 바둑알 저장 자료구조)
-게임 진행 현황 객체(착수 차례(바둑알 정보 객체), 착수 회수, 착수 좌표정보)
-통합객체(커서, 바둑판, 게임 진행 현황 객체)

3.2. omokobj_manage.c / omokobj_manage.h
오목 프로그램 객체를 관리(생성, 조작, 제거)하는 함수 정의 및 구현

3.3. omokobj_conio.c / omokobj_conio.h
자연스러운 콘솔 출력에 필요한 Windows.h 함수를 정의한 뒤
오목 프로그램 객체를 콘솔에 출력하는 함수 정의 및 구현

3.3. omok_algorithm.c / omok_algorithm.h
오목 규칙(33, 44) 준수여부, 승패 여부를 판단하는 함수 정의 및 구현

3.4. omok.c / omok.h
오목 프로그램을 실행 및 종료하고 오목 관련 함수를 호출하는 함수 정의 및 구현

3.5. omok_err.c / omok_err.h
오목 프로그램 실행 중 발생하는 에러를 관리하는 함수 정의 및 구현

3.5. main.c
오목 실행 함수를 호출하는 메인 함수
